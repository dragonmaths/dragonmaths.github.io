Multiplying by Tens

When working beyond nine the counting is done in tens.
ten Tens is one Hundred.
ten Hundreds is one Thousand
ten Thousands is one Ten Thousand.
hundred Hundreds is one Ten Thousand.

In terms of multiplication:
10x10=100
10x100=1000
10x1000=10000
100x100=10x10x100=10x1000=10000

A technique when multiplying in tens, hundreds, thousands and so is is to count the trailing zeros.
Each 0 positions the 1 in the correct column
The number of 0s in the multiplication matches the number of 0s in the result;

30x20 = 3x10x2x10 = 3x2x10x10 = 6x100 = 600 since multiplying order does not matter
Using the counting 0s technique there are two trailing 0s in 30x20 so two in answer
So calculate 3x2 = 6 and apply the correct number of 0s (two) to position the 6
So 30x20 = 600

Another example
500x70
Using the counting 0s technique there are three trailing 0s in 500x70 so three in answer
So calculate 5x7 = 35 and apply the correct number of 0s (three) to position the 35
So 500x70 = 35000

Taking Away, Minus, Negative and Other Words

In this treatise (-1) is called Minus One and is a result.
The action – is called taking away.
On the Taking Away page and equating actions with results it was shown that the action Taking Away One and the result Minus One are equivalent.
Often in Maths rather than taking away, minus is used as the same action as in 5 minus 3 is 2
Hence –(-1) is often said as minus minus one.
In this case strictly the two uses of minus are different - the first is an action (taking away) the second as part of a name of a number (Minus One)

Why the name Minus One in the first place?
Well when minus is used for the action taking away and since –1 = -1 then minus (the action) One gives a result which is called Minus One from the action that led to it.
-1 could have been called Taking Away One but that would have been a bit of a mouthful.
In proper mathematical circles -1 is called Negative One and taking away is called subtract.

There is another way to think of Minus One using Minus as a different action (which is taking the negative)
1 is and is red.
(-1) is and is blue.
Minus can be thought of as the action swap colours. So red goes to blue and blue to red.
(-1) can be thought of as start with the number One (red) and swap it for blue.
So Minus One is the (blue) number obtained when the (red) number One is switched to blue.

Using Minus to mean swap then
Minus Minus One could mean start with One (red) swap (first Minus) to Minus One (blue) and then swap (second Minus) to red which is One.
Minus Minus One could mean start with Minus One (blue) and swap (second Minus) to red which is One. Same result
So -(-1) = –(-1) = 1
So Minus Minus One, take away Minus One and One are taken as the same equating actions and results.
